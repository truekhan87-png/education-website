# EduLearn Academy

## Description

EduLearn Academy is a static educational website created using HTML and CSS.

The website includes:
- Home Page
- About Section
- Courses Section
- Why Choose Us Section
- Contact Form
- Responsive Design

This project was developed as Frontend Development Project 1.

---

## Technologies Used

- HTML5
- CSS3

---

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click on `index.html`
   OR
4. Open the project in VS Code and use Live Server.
